When tagging maybe and almost valid cards, assume that either the card works the way you think it might, or the necessary changes have been made to make the card valid.

  • ATROCIOUS: The card is just terrible (e.g. a vanilla 1/1 for CMC 6)
  • BROKEN: The card will break the game if played. This generally includes cards that create infinite loops or easily enable infinite loops.
  • FUNNY: Name, text, or some other combination of the card's features are comical
  • NOVEL: The card is generally balanced and has some unique or interesting rules text that could create some fun strategies and gameplay. This tag can be used to prioritize what cards should go in the cube.
  • OVERPOWERED: The value you get from playing the card is way higher than its cost. A step below broken, but still way better than it probably should be. (e.g. a vanilla 6/6 for CMC 3)
  • PLAGIARIZED: Sometimes the neural net will lift the name or rules text (or both) verbatim from an already existing card. Don't worry about checking the card now, if you think any part of it might have been plagiarized, tag it as such and we can check it later if we're considering adding it to the cube.
  • UNLIKELY: The card needs a very specific and unlikely situation in order to be useful. May often be atrocious.
  • YOU TRIED: The card starts off fine, but then goes off the rails towards the end. Or, the card has a large amount of complex rules text that is almost all valid, but one part of it invalidates the whole thing.
